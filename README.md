-----

## 📋 السياق والمتطلبات الأساسية

أنت مطور Node.js خبير متخصص في بناء بوتات لمنصة WOLF Live باستخدام مكتبة `wolf.js` (الإصدار 2.6.2) من مستودع `https://github.com/dawalters1/wolf.js.git`.

**المطلوب:** برمجة بوت لعبة الجاسوس (Spy Game Bot) في **ملف JavaScript واحد فقط** (`bot.js`) يعمل على منصة WOLF Live.

## 🎮 وصف اللعبة الكاملة: “لعبة الجاسوس”

### المفهوم الأساسي:

لعبة اجتماعية تفاعلية يُرسل فيها البوت كلمة سرية (اسم فاكهة مختارة عشوائياً) لجميع اللاعبين **عبر الرسائل الخاصة** باستثناء لاعب واحد يُختار عشوائياً ليكون **الجاسوس**. يحاول اللاعبون كشف الجاسوس بينما يحاول الجاسوس إخفاء هويته.

## 📝 المتطلبات التفصيلية

### 1️⃣ إنشاء اللعبة

**أوامر الإنشاء (في محادثة المجموعة/الغرفة):**

- `!جاسوس جديد`
- `!جس جديد`
- `!spy new` ← هذا الأمر يُعيِّن لغة اللعبة **إنجليزية**

**رسالة البوت بعد الإنشاء (عربي):**

```
/me يلا يا حلوين بدينا اللعبه انظموا للعبه بالأمر هذا "!جاسوس انظم او !جس انظم"
```

**رسالة البوت بعد الإنشاء (إنجليزي - عند استخدام `!spy new`):**

```
/me Come on, sweeties, we've started the game. Join the game with this command: "!spy join"
```

**قواعد الإنشاء:**

- لا يمكن إنشاء لعبة جديدة إذا كانت هناك لعبة نشطة بالفعل
- يُسجَّل مُنشئ اللعبة (sourceId) لاستخدامه لاحقاً في التحقق من الصلاحيات
- تُحدَّد لغة اللعبة (عربي/إنجليزي) عند الإنشاء وتُستخدم في جميع رسائل البوت

-----

### 2️⃣ الانضمام للعبة

**أوامر الانضمام:**

- `!جاسوس انظم`
- `!جس انظم`
- `!spy join` (للغة الإنجليزية)

**القواعد:**

- يمكن الانضمام فقط بعد إنشاء اللعبة وقبل بدئها
- لا يمكن للاعب الانضمام مرتين
- يُضاف اللاعب إلى قائمة اللاعبين بـ: `{ id, nickname, points: 0 }`
- الحد الأدنى للانضمام: **2 لاعبين** لبدء اللعبة

-----

### 3️⃣ بدء اللعبة

**أوامر البدء (من مُنشئ اللعبة فقط):**

- `!جاسوس بدء`
- `!جس بدء`
- `!spy start` (للغة الإنجليزية)

**عند بدء اللعبة يقوم البوت بما يلي بالترتيب:**

#### أ) اختيار الفاكهة السرية عشوائياً من القائمة المرفقة

#### ب) اختيار الجاسوس عشوائياً من بين اللاعبين

#### ج) إرسال كلمة السر للاعبين عبر رسائل خاصة (ما عدا الجاسوس):

```
/alert كلمة السر هي: [اسم الفاكهة] 🍎
```

(إنجليزي):

```
/alert The secret word is: [fruit name] 🍎
```

#### د) إرسال رسالة خاصة للجاسوس:

```
/alert انت الجاسوس يا قلب قلبي اخدعهم واختار أي لاعب من القائمه عشان محد يشك فيك 🥴
```

(إنجليزي):

```
/alert You are the spy, sweetheart! Deceive them and choose any player from the list so no one suspects you 🥴
```

#### هـ) إرسال قائمة اللاعبين في المحادثة العامة:

```
🕵️ قائمة اللاعبين:
1- [ID] | [اسم اللاعب]
2- [ID] | [اسم اللاعب]
3- [ID] | [اسم اللاعب]
...
```

(إنجليزي):

```
🕵️ Players List:
1- [ID] | [nickname]
2- [ID] | [nickname]
...
```

-----

### 4️⃣ جولة التصويت والمراهنة

#### التصويت:

بعد عرض قائمة اللاعبين، يقوم كل لاعب مُنضم في اللعبة بإرسال **رقم اللاعب** الذي يعتقد أنه الجاسوس مباشرةً في المحادثة العامة.

- يُقبل الرقم فقط من اللاعبين المسجلين في اللعبة
- يصوت الجاسوس أيضاً لأي رقم للتمويه
- لا تنتهي الجولة إلا بعد تصويت **جميع** اللاعبين

#### المراهنة (اختياري):

يمكن للاعبين المراهنة **قبل أو أثناء** التصويت بالأمر:

- `!جاسوس [رقم اللاعب] [كمية النقاط]`
- `!جس [رقم اللاعب] [كمية النقاط]`
- `!spy [player number] [amount]` (للغة الإنجليزية)

**مثال:** `!جاسوس 3 50` ← يراهن اللاعب بـ 50 نقطة على أن اللاعب رقم 3 هو الجاسوس

**قواعد المراهنة:**

- الجاسوس يمكنه المراهنة على أي رقم للتمويه
- من يصيب: يكسب **ضعف** ما راهن به
- من يخطئ: يخسر **ضعف** ما راهن به
- لا يمكن المراهنة بأكثر من رصيد النقاط الحالي

-----

### 5️⃣ كشف الجاسوس

بعد تصويت **جميع** اللاعبين، يُرسل البوت في المحادثة العامة:

**عربي:**

```
/alert هذا هو الخاين البواق:
[ID الجاسوس] | [اسم الجاسوس]
```

**إنجليزي:**

```
/alert This is the traitor:
[Spy's ID] | [Spy's nickname]
```

-----

### 6️⃣ حساب النقاط

**السيناريوهات:**

|الموقف                        |النتيجة                             |
|------------------------------|------------------------------------|
|لاعب صوّت بشكل صحيح على الجاسوس|+1 نقطة للمصوِّت                      |
|لاعب صوّت بشكل خاطئ            |0 تغيير                             |
|الجاسوس: كل لاعب تمكن من كشفه |-1 نقطة من الجاسوس لكل لاعب كشفه    |
|نقاط الجاسوس تصل للسالب       |تُسجَّل بالسالب (مسموح بالنقاط السالبة)|
|مراهنة صحيحة                  |+ضعف المبلغ المراهن                 |
|مراهنة خاطئة                  |-ضعف المبلغ المراهن                 |

-----

### 7️⃣ الاستمرار أو إنهاء اللعبة

بعد كشف الجاسوس، يُرسل البوت في المحادثة العامة:

**عربي:**

```
/alert اذا ودك تكمل اللعبه ارسل رقم 1 او اذا مالك خاطر ارسل رقم 2
```

**إنجليزي:**

```
/alert If you want to continue the game, send the number 1, or if you don't want to, send the number 2.
```

- يُقبل الاختيار **من مُنشئ اللعبة فقط**
- `1` → تبدأ جولة جديدة (يُختار جاسوس جديد وفاكهة جديدة)
- `2` → تُغلق اللعبة نهائياً

|الحالة                            |المدة  |الإجراء                       |
|----------------------------------|-------|------------------------------|

|انضم لاعبون لكن لم تبدأ اللعبة    |3 دقائق|إغلاق تلقائي                  |
|بدأت اللعبة لكن لم يصوّت أحد       |3 دقائق|إغلاق تلقائي                  |
|بعض اللاعبين لم يصوتوا بعد 2 دقيقة|2 دقيقة|طرد غير المصوتين وإكمال اللعبة|

## ⚙️ متطلبات التنفيذ التقني

### 1. هيكل الكود في ملف واحد `bot.js`:

```
bot.js
├── imports ومتغيرات الإعداد
├── قائمة الفواكه (fruits array)
├── إدارة حالة اللعبة (gameState object)
├── نظام النقاط (points management + JSON persistence)
├── دوال مساعدة (helper functions)
│   ├── getRandomFruit()
│   ├── getRandomSpy(players)
│   ├── sendPrivateMessage(client, userId, message)
│   ├── sendGroupMessage(client, groupId, message)
│   ├── buildPlayersList(players, language)
│   ├── resetGame()
│   └── clearTimers()
├── معالجات الأوامر (command handlers)
│   ├── handleNewGame(command, language)
│   ├── handleJoin(command)
│   ├── handleStart(command)
│   ├── handleVoteAndBet(command)
│   ├── handleKick(command)
│   ├── handleRank(command)
│   ├── handleGeneral(command)
│   ├── handleTotal(command)
│   ├── handleHelp(command, language)
│   └── handleContinue(command)
├── إعداد معالجات الأحداث (event listeners)
└── تسجيل الدخول وتشغيل البوت
```

### 2. متطلبات wolf.js API:

- استخدم `client.on('message', handler)` لمعالجة الرسائل
- استخدم `client.messaging.sendMessage(targetType, targetId, message)` لإرسال الرسائل
- `targetType`: `'group'` للمجموعات، `'private'` للرسائل الخاصة
- استخدم `message.sourceId` لمعرفة هوية المرسل
- استخدم `message.targetId` لمعرفة الغرفة/المجموعة
- استخدم `message.content` أو `message.body` لمحتوى الرسالة
- تسجيل الدخول: `client.login(email, password)`

### 3. استمرارية البيانات:

- احفظ نقاط اللاعبين في ملف `points.json` بجانب `bot.js`
- استخدم `fs.readFileSync` و `fs.writeFileSync` لقراءة/كتابة البيانات
- احمِ من أخطاء JSON باستخدام try/catch

### 4. معالجة الأخطاء:

- تعامل مع انقطاع الاتصال وإعادة الاتصال تلقائياً
- أضف console.log لتسجيل الأحداث المهمة
- تعامل مع الحالات الحافية (لاعب واحد، رسائل غير متوقعة)

## ⚠️ تعليمات مهمة للمطور

1. **الملف الواحد:** كل الكود يجب أن يكون في `bot.js` فقط - لا ملفات إضافية عدا `points.json`
1. **التحقق من الصلاحيات:** فقط مُنشئ اللعبة يمكنه: البدء، الطرد، الاستمرار/الإيقاف
1. **العشوائية:** استخدم `Math.random()` لاختيار الفاكهة والجاسوس في كل جولة
1. **اللغة:** يتم تحديد لغة اللعبة مرة واحدة عند الإنشاء وتستمر حتى انتهاء اللعبة
1. **المؤقتات:** استخدم `setTimeout` و `clearTimeout` وتأكد من مسح المؤقتات عند إعادة تعيين اللعبة
1. **الرسائل الخاصة:** يُرسل البوت كلمة السر ورسالة الجاسوس عبر **رسائل خاصة** فقط
1. **النقاط السالبة:** مسموح للجاسوس بامتلاك نقاط سالبة
1. **عدم التكرار:** اختر فاكهة مختلفة في كل جولة (يمكن إعادة الاختيار بعد استنفاد القائمة)
